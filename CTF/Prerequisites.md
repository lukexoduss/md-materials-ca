# Math

## **Web Exploitation (WE)**
- **Minimal math required**
- Basic understanding of: HTTP protocols, encoding (Base64, URL encoding), sometimes basic logic/Boolean operations
- More focused on understanding web technologies than mathematics

## **Cryptography (Cr)**
- **Heavy math requirements**
- Number theory (modular arithmetic, prime numbers, greatest common divisors)
- Abstract algebra (groups, fields, rings)
- Probability and statistics
- Linear algebra (for some modern crypto)
- Discrete mathematics

## **Reverse Engineering (RE)**
- **Light to moderate math**
- Binary/hexadecimal number systems
- Bitwise operations (AND, OR, XOR, shifts)
- Basic discrete math
- Understanding of computer architecture concepts

## **Digital Forensics (DF)**
- **Minimal math required**
- File system structures and data representation
- Hash functions (conceptual understanding)
- Possibly basic statistics for timeline analysis

## **Binary Exploitation (Pwn) (BE)**
- **Light to moderate math**
- Binary/hexadecimal arithmetic
- Memory address calculations
- Bitwise operations
- Understanding of number representations (signed/unsigned integers)

## **Steganography (S)**
- **Moderate math**
- Bit manipulation
- Sometimes linear algebra (for image-based techniques)
- Basic statistics (for detection)
- Signal processing concepts (for audio/image steganography)

## **OSINT**
- **Minimal math required**
- Primarily research and analytical skills
- Possibly basic statistics for data analysis

## **Network Traffic Analysis (NTA)**
- **Minimal to light math**
- Understanding of network protocols
- Sometimes basic statistics for anomaly detection
- Binary/hexadecimal for packet analysis

## **Password Cracking (PC)**
- **Light math**
- Understanding of hash functions (conceptual)
- Combinatorics (for calculating keyspace)
- Probability (for attack efficiency)

## **Digital Forensics and Incident Response (DFIR)**
- **Minimal math required**
- Similar to Digital Forensics
- Timeline analysis
- Hash functions (conceptual)

**General foundation helpful for all areas:**
- Boolean logic
- Basic understanding of binary and hexadecimal systems
- Fundamental computer science concepts

The math intensity roughly ranks: **Cryptography >> Steganography > Binary Exploitation/Reverse Engineering > Password Cracking > Others**
